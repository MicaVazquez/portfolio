# Portfolio — Micaela Vázquez

Mi portfolio personal como Desarrolladora Full Stack.

**🌐 Ver online: [portfolio-micaelavazquez.vercel.app](https://portfolio-micaelavazquez.vercel.app/)**

## Tecnologías

- [Next.js](https://nextjs.org) (App Router) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com)
- Tipografías: Space Grotesk y Geist Sans vía `next/font`
- Deploy automático en [Vercel](https://vercel.com)

## Correr en local

```bash
pnpm install
pnpm dev
```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
